# DSA (Data Structures and Algorithms)

This repository contains my implementations and practice problems for Data Structures and Algorithms using Java. The goal of this repository is to improve problem-solving skills and build a strong foundation in algorithmic thinking.

---

## Topics Covered

* Arrays
* Strings
* Linked Lists
* Stacks
* Queues
* Recursion
* Searching Algorithms
* Sorting Algorithms
* Greedy Algorithms
* Dynamic Programming
* Trees
* Graphs
* Hashing
* Backtracking

